Enables Virtu Systems mortgage widget and shortcode.

== Description ==
This plugin adds the required javascript for Virtu mortgage widget.

For more information visit: [Virtu Systems](https://virtusystems.ru/)

== Installation ==
1. Upload `simple-virtu-widget` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place shortcode `[virtu-widget /]` where you want the widget to be displayed.
